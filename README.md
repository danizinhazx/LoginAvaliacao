Trabalo 5 Avaliação - App de Login

O projeto consiste em um aplicativo Android nativo (Kotlin) que implementa um fluxo de autenticação completo, seguindo as diretrizes de Interface (XML) e Lógica (Kotlin) propostas.

Alunas:
Daniela Souza de Almeida e Laís Pereira Silva

Funcionalidades Implementadas
1.  Validação de Campos:
    Verificação automática se o e-mail contém "@".
    Verificação se a senha possui no mínimo 6 caracteres.
    Exibição de erros visuais nos campos (`TextInputLayout`).
2.  Fluxo de Login:
    Comparação com credenciais fixas no código.
    Navegação para a `HomeActivity` apenas em caso de sucesso.
